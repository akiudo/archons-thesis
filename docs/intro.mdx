# The Archon's Thesis

Welcome to **The Archon's Thesis** — an interconnected lore encyclopedia and
companion to the FFXIV Main Scenario Quest.

<LorePanel variant="lore" title="About This Project">

This project functions as an authoritative, accessible narrative guide for players
who want deep world context without extensive narrative gaps. Content is organized
along the MSQ timeline and enriched with primary source documentation.

</LorePanel>
