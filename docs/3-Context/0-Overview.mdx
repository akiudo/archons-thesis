# Context

**Context** holds explanations the MSQ does not spell out — gaps between cinematics
and gameplay, legacy history your character never lived, and other clarifications
that help the story make sense without advancing the quest line.

<LorePanel variant="lore" title="Optional Reading">

Context entries are not required to follow the MSQ. Read them when a Journey chapter
points you here, or when something in the game leaves you wondering what you missed.

</LorePanel>
