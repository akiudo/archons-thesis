# The Journey

**The Journey** follows the Main Scenario Quest beat by beat. Each chapter is
anchored to a narrative milestone you reach in the game — read it when you get there,
not before.

<LorePanel variant="lore" title="Read in Step with the MSQ">

Start with the first chapter in the sidebar and return here as you progress through
*A Realm Reborn*. Entries stay within spoiler boundaries for where you likely are in
the story.

</LorePanel>

[Context](/docs/A-Realm-Reborn/Context). For thematic world-building outside the
