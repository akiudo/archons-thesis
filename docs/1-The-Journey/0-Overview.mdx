# The Journey

**The Journey** follows the Main Scenario Quest beat by beat. Each chapter is
anchored to a narrative milestone you reach in the game — read it when you get there,
not before.

<LorePanel variant="lore" title="Read in Step with the MSQ">

Choose your expansion in the sidebar and read chapters as you progress. Entries stay
within spoiler boundaries for where you likely are in the story.

</LorePanel>

Need background on something the MSQ skips over? See
[Context](/docs/Context). For thematic world-building outside the
quest line, see [The Encyclopaedia](/docs/The-Encyclopaedia).
