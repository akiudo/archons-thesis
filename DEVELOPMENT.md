# Development

This site is built with [Docusaurus](https://docusaurus.io/). Node.js 20+ is required.

## Install

```bash
npm install
```

## Local Development

```bash
npm run start
```

This starts a local development server with hot reload. Most changes are reflected immediately without restarting the server.

## Build

```bash
npm run build
```

This generates static content into the `build/` directory.

To preview the production build locally:

```bash
npm run serve
```

## Deployment

Deployment to GitHub Pages runs automatically on push to `main` via GitHub Actions.
