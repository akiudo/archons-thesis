# Contact

Have a correction, a question, or a lore suggestion? Here is how to reach us.

<LorePanel variant="lore" title="Get in Touch">

- **GitHub** — Report issues, suggest entries, or contribute via the
  [project repository](https://github.com/controvi/archons-thesis).
- **Corrections** — Spotted an error or a spoiler that slipped through? Open a
  GitHub issue with the entry name and details.

</LorePanel>

## Contributing

The Archon's Thesis is a living project. If you would like to help with research,
writing, or citations, open an issue on GitHub describing what you would like to
work on.

## Feedback

Feedback on pacing, spoiler handling, and clarity is especially welcome — those are
the hardest parts of a project like this to get right, and player input helps.
