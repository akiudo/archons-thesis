# The Archon's Thesis

A narrative companion to the Final Fantasy XIV Main Scenario Quest.

The Archon's Thesis is a lore encyclopedia and companion project for *Final Fantasy XIV*. It follows the Main Scenario Quest timeline as a chronological guide, pausing at key story milestones to provide context, thematic deep-dives, and references to primary sources — without getting ahead of where you are in the story.

**Live site:** [archons-thesis.com](https://archons-thesis.com)

## Corrections & Feedback

This project is a work in progress. If you spot a factual error, typo, broken link, or anything that looks wrong, please [open an issue](https://github.com/controvi/archons-thesis/issues/new) on GitHub. A short description of what you found and where you saw it is enough.
