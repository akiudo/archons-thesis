## Lore Index

I have added a new [Lore Index](/lore-index) page that you can use to search for
terms, names, places, and more. This index will gradually fill out over time, making
it easy to find information on exactly what you are looking for.

Because FFXIV is so story-heavy, I built a specific system to keep everything
spoiler-free. By default, the index starts in an "everything is a spoiler" setting.
You can manually open up the spoiler tags for each specific expansion and patch. Be
sure to use this feature so you don't accidentally read anything ahead of your
current place in the playthrough!

## Updated Homepage

The [homepage](/) has also received a restructure to provide a much better flow to
the content. You can still find information on how to use the site using the same
links as before, but you can now also jump straight into the lore using new direct
links.

## Upcoming Content

Currently, I am working on the lore of Gridania, which should be finished later this
week. After that, I will be writing about the history of Ul'dah.

It is fascinating how much lore exists for what initially looks like a simple town
in a forest. The history behind this place is extensive, so my goal right now is to
keep it as concise as possible so it doesn't get too long. I will likely add more
deep-dive details later on, but for now, I am doing my best to make it a great,
accessible read.
